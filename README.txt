# Meta Title & Description

## Description
A simple WordPress plugin that adds **Meta Title** and **Meta Description** fields to every post and page. It allows you to manage SEO metadata directly from the editor.

## Features
- Metabox with Title and Description fields for posts and pages.
- Saves data into post meta fields.
- Automatically outputs `<title>` and `<meta name="description">` tags in the `<head>` on the frontend.

## Installation
1. Download the plugin and upload it to the `/wp-content/plugins/meta-title-description/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
